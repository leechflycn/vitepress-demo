# 飞书机器人 (急救龙虾) 注册与配置全通关指南

这份指南将手把手教你如何从零开始注册飞书、创建企业，并配置一个专属的「急救龙虾」机器人（接入 OpenClaw 等 AI 框架）。全程免费，无需公网 IP。

## 第一阶段：注册飞书与创建测试企业

如果你已经有飞书账号和企业，可以跳过这一步。

1. **下载与注册**：前往 [飞书官网](https://www.feishu.cn/) 下载客户端或直接在网页版用手机号注册。
2. **创建企业/团队**：
   - 登录后，系统通常会提示你加入或创建企业。选择**创建企业**。
   - 随便起个名字（比如“龙虾极客工作室”），行业规模随便选。飞书的基础版对小团队和个人是完全免费的。
   - 创建完成后，你就是这个企业的超级管理员。

## 第二阶段：在开放平台创建机器人应用

1. **登录开放平台**：使用刚刚的飞书账号，登录 [飞书开放平台 (open.feishu.cn)](https://open.feishu.cn/app)。
2. **创建自建应用**：
   - 点击 **「创建企业自建应用」**。
   - **应用名称**：填写「急救龙虾」（或其他你喜欢的名字）。
   - **应用描述**：随便写，比如“我的私人 AI 助理”。
   - **应用图标**：上传一个帅气的龙虾头像。
   - 点击创建。

## 第三阶段：核心配置与避坑（重点）

进入应用详情页后，跟着下面这几步走，顺序不要乱。

### 1. 添加应用能力
- 在左侧菜单找到 **「添加应用能力」**。
- 找到 **「机器人」**，点击添加。这就是我们能和它聊天的基础。

### 2. 权限管理（极其重要，必看避坑指南）
在左侧菜单找到 **「权限管理」**。你需要给机器人赋予读写消息的权限。
在搜索框中搜索并勾选以下核心权限：
- `im:message` (获取与发送单聊、群组消息)
- `im:chat` (获取群组信息)
- `im:resource` (获取消息中的图片、文件)
- *其他根据你需要使用的扩展能力酌情添加（比如飞书文档权限 `docs:doc`、云文档权限 `drive:drive` 等）。*

⚠️ **【史诗级巨坑警告】**：在飞书里，**所有权限的修改，必须通过发布新版本才会生效**！勾选完权限后，千万别忘了去发布（详见第4步）。

### 3. 事件与回调配置 (长连接模式)
为了让机器人能实时收到你的消息，且**不需要你提供公网 IP**（非常适合部署在本地 PVE 或群晖里）：
- 左侧菜单找到 **「事件与回调」**。
- 将回调方式切换为 **「WebSocket 长连接」** 模式（不要选 Webhook，Webhook 需要公网域名配置）。
- 订阅以下核心事件：
  - `im.message.receive_v1` (接收消息事件)

### 4. 版本发布与审核
这是最容易卡住新手的一步：
- 左侧菜单找到 **「应用发布」** -> **「版本管理与发布」**。
- 点击 **「创建版本」**。
- 版本号随便填（如 `1.0.0`），更新说明写“初始发布”。
- **可用范围**：选择你自己，或者整个企业的所有人。
- 点击**保存并申请发布**。
- 因为你是企业管理员，申请后系统会**自动秒过审核**。发布上线后，你刚才配的权限和长连接才会真正生效！

## 第四阶段：获取凭证并填入框架

最后一步，拿到机器人的“身份证”和“密码”，填到你的 OpenClaw (或其他 AI) 配置文件里。

1. 左侧菜单回到 **「凭证与基础信息」**。
2. 找到 **App ID** 和 **App Secret**。
3. 复制这两串字符。
4. 在你的 OpenClaw 的 `feishu` 通道配置中填入：
   ```yaml
   channels:
     feishu:
       appId: "cli_a1b2c3d4e5f6g7h8"
       appSecret: "xxxxx_your_secret_xxxxx"
   ```
5. 启动你的程序。

## 第五阶段：在飞书中开始聊天

1. 打开飞书客户端的搜索栏。
2. 搜索你刚才创建的应用名称「急救龙虾」。
3. 点击它，就可以开始私聊了！
4. 你也可以拉个群，在群设置的“群机器人”里把它添加进去，然后在群里 `@急救龙虾` 召唤它。
