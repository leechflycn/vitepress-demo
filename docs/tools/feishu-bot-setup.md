# 飞书机器人 (急救龙虾) 注册与配置全通关指南（图文保姆级）

这份指南将手把手、图文并茂地教你如何从零开始注册飞书，并配置一个专属的「急救龙虾」机器人。全程免费，不需要你懂代码，也不需要公网 IP！

---

## 第一步：注册飞书与创建企业

1. **打开官网**：首先在浏览器打开 [飞书官网](https://www.feishu.cn/)。
2. **手机号登录**：点击右上角的“登录”，直接用你的手机号接收验证码登录。
3. **创建企业**：
   - 登录后，系统会问你是加入现有企业还是创建新企业。
   - 请点击 **“创建企业”**。
   - **企业名称**随便填（比如：龙虾极客工作室），行业和规模随便选（比如：IT / 1-10人）。
   - 飞书的基础版是永久免费的，放心创建。

> 💡 **提示：** 如果你已经有飞书企业了，这步可以直接跳过。

## 第二步：在开放平台创建你的“机器人应用”

1. **进入开发者后台**：在浏览器新开一个标签页，打开 [飞书开放平台后台 (open.feishu.cn/app)](https://open.feishu.cn/app)。
2. **创建自建应用**：
   - 在页面左侧，点击 **「创建企业自建应用」** 这个大按钮。
3. **填写应用信息**：
   - **应用名称**：写上「急救龙虾」或者你想取的名字。
   - **应用描述**：填“我的私人 AI 助理”。
   - **应用图标**：上传一张你喜欢的图片作为机器人的头像。
   - 点击右下角的 **「创建」** 按钮。

## 第三步：给应用添加“机器人”灵魂

刚创建好的应用只是个空壳，我们需要赋予它接收和发送消息的能力。

1. 在应用的管理后台，看 **左侧菜单栏**。
2. 点击 **「添加应用能力」**。
3. 在右侧出现的列表中，找到 **「机器人」**，然后点击它下面的 **「添加」** 按钮。

## 第四步：配置核心权限（🔥极易踩坑，必看！）

机器人要能看懂你发的消息、能给你发图片，必须要有相应的“权限”。

1. 看左侧菜单栏，点击 **「权限管理」**。
2. 在页面的搜索框里，依次搜索并**打上勾**。对于日常手机聊天、群组互动以及文档处理，**以下是核心必开权限（缺一不可）**：

   **【核心聊天能力】**
   - `im:message:send_as_bot`（获取单聊、群组消息 —— **最重要，必须开！**）
   - `im:message.p2p_msg:readonly`（接收单聊消息）
   - `im:message.group_at_msg:readonly`（接收群聊中@我的消息）
   - `im:message:readonly`（获取消息内容）
   - `im:message`（获取和发送单聊、群组消息）
   - `im:resource`（获取消息中的图片、文件）
   
   **【如果你需要在群里拉它】**
   - `im:chat`（获取和更新群组信息）
   - `im:chat:read`（读取群组信息）
   
   **【如果你需要让它处理云文档（可选）】**
   - `docx:document`（读写文档）
   - `drive:drive`（云盘文件读写）

⚠️ **【超级避坑提示】**：勾选完之后，这里只是暂存，**必须通过第六步的“发布新版本”才会真正生效**！新手最容易卡在这里。

## 第五步：开启 WebSocket 长连接（免公网 IP 的秘密）

为了让机器人能秒回你的消息，而且不需要你去搞复杂的公网 IP 和域名穿透，我们需要开启 WebSocket 模式。

1. 看左侧菜单栏，点击 **「事件与回调」**。
2. 在页面上方，找到“回调方式”，把它切换到 **「WebSocket 长连接」**（注意：千万不要选 Webhook）。
3. 接着在同一个页面，点击 **「添加事件」**。
4. 搜索 `im.message.receive_v1`，勾选它（中文名叫：接收消息 v1.0）。

## 第六步：发布应用（让一切配置生效）

刚才配的权限和长连接，如果不发布，机器人就还是个“聋子”和“哑巴”。

1. 左侧菜单栏，向下滚动找到 **「应用发布」**，点击 **「版本管理与发布」**。
2. 点击右上角的 **「创建版本」** 按钮。
3. **应用版本号**：随便填，比如填 `1.0.0`。
4. **更新说明**：随便填，比如填 `初始发布`。
5. **可用范围**：选择你自己的名字，或者直接选“全部成员”。
6. 点击最下方的 **「保存并申请发布」**。
7. 因为你就是这个企业的超级管理员，系统会直接提示你审核通过，状态变为 **已上架**。

## 第七步：拿到机器人的“身份证”并启动配置

1. 左侧菜单栏回到最上面，点击 **「凭证与基础信息」**。
2. 在页面中间，你会看到两个很重要的东西，点击它们旁边的小眼睛显示出来：
   - **App ID** (类似账号，一般是 `cli_` 开头)
   - **App Secret** (类似密码，很长一串)
3. 把这两串字符复制下来。
4. 填入到你的 OpenClaw (或其他 AI 框架) 的配置文件中：

```yaml
channels:
  feishu:
    appId: "把你的App ID粘贴到这里"
    appSecret: "把你的App Secret粘贴到这里"
```

## 最后：去飞书找它聊天！

1. 打开你手机或电脑上的 **飞书客户端**。
2. 在最上面的搜索框里，搜索「急救龙虾」（或者你起的名字）。
3. 看到它的头像，点进去，你就可以直接和它聊天了！
4. **进阶玩法**：你还可以建一个群，在群设置 -> 群机器人 里把它拉进去，以后在群里 `@急救龙虾` 它就会在群里回复你了。
